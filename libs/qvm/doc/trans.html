Boost QVM

trans

#include <boost/qvm/map_mv.hpp>

namespace boost
{
    namespace qvm
    {
        //Only enabled if:
        //  is_m<A>::value &&
        //  m_traits<A>::rows==m_traits<A>::cols && m_traits<A>::rows>=3
        template <class A>
        -unspecified-return-type- trans( A & a );
    }
}

The expression trans(m) returns a view proxy that accesses the translation component of the square matrix m, which is a vector of size D-1, where D is the size of m.

See also: boost/qvm/map_mv.hpp
