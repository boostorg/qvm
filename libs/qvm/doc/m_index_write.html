Boost QVM

m_index_write

#include <boost/qvm/m_index.hpp>

namespace boost
{
    namespace qvm
    {
        //Only enabled if:
        //  is_m<A>::value
        template <class A>
        typename m_traits<A>::scalar_type &
        m_index_write( A & a, int r, int c );
    }
}

Returns:

A mutable reference to the element of a at row r and column c.

Throws:

In case either r or c is out of bounds, throws matrix_index_out_of_bounds_error.

See also: boost/qvm/m_index.hpp
